# Senra-et-al-BRAGANTIA-2026
This repository hosts the supplementary data supporting the article published in Bragantia, “Path analysis aiding the early selection of Conilon coffee for multiple environments”. 
It includes further data gathered with the analyses, aiming to ensure transparency, reproducibility, and data reuse in coffee breeding research.

Senra, J. F. B., Rodrigues, W. N., Colodetti, T. V., Ferreira, A., Esposti, M. D. D., Milheiros, I. S., Silva, T. S. and Ramos, I. B. (2026). Path analysis aiding the early selection of Conilon coffee for multiple environments. Bragantia, xx, exxxx. https://doi.org/
